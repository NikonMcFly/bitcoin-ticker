# bitcoin-ticker
Learning ES6 and React Redux with bitcoin ticker [Bitcoin Average](bitcoinaverage.com) API


## Run Locally
1. `npm install`
2. start webpack `webpack --watch`
3. open another terminal visit `http://localhost:8080`
